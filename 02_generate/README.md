# 02_generate


To run:

- Generate a go module (`go.mod`)
- Add the pixelart package via go get to `go.mod`
- Run the code


```
$ cd 02_generate
$ go mod init example.com/02_generate
$ go get github.com/pixelartexchange/artbase.server/pixelart
$ go run main.go
```
